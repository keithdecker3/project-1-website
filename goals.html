2/9/18 Goals:

  • Create a Habitica character to help create goals and accountability
  • Learn about firebase and submit at least one drill
  • Learn at least two new things that javascript can do.

Reflection:

I completed all goals

I started the day by making a habitica character to help set goals. I set a lot of habits that revolve around the bootcamp such as submitting drills, helping classmates, studying vs procrastinating, etc. I also set some personal goals such as meditation and stretching. Having a character to level up based on completing goals has already been surprisingly fun. Firebase was pretty straightforward when I had the practice up to follow along with. After the first two I challenged myself to launch the last example site based on memory alone. I made a couple mistakes and had to restart but I believe that it helped me retain the knowledge better. I ran through some of the practice material for javascript after that. Some of it was familiar from precourse work and self-teaching but objects were pretty new to me. I learned how to create objects in javascript and access values using dot notation and square bracket notation.

Overall I would say that finding a fun way to track goals and create accountability was a solid win today. I also learned that resources are great but if I don't challenge myself to use my memory that can lead to less retention of the material which is what I saw happening with the firebase drills. I learned that I tend to try and write all of the code for a javascript challenge at once which makes it hard to troubleshoot if it doesn't work. I learned that using something like repl.it and comments allows me to isolate the parts of the code that I don't understand and be more intentional with what I am learning.
